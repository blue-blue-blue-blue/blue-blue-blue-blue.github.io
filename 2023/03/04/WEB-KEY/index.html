网站源码

当开发人员在线上环境中对源代码进行了备份操作，并且将备份文件放在了 web 目录下，就会引起网站源码泄露。
通过Python 来遍历一下这个网页的子页面，（状态码200 成功处理了请求，一般情况下都是返回此状态码），如果存在我们就可以网络访问这个页面，拿到相应的信息。

查看网页源码的方法:

  • F12
  • Ctrl+Shift+I
  • 鼠标右键——查看源码
  • 鼠标点右上角，调出开发者工具
  • Ctrl+U
  • view-source:url
